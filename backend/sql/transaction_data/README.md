# トランザクションデータ

このディレクトリには、開発・テスト用のサンプルデータを含むSQLファイルが格納されています。

## 注意事項

- 本番環境では実行しないでください
- ユーザーが画面から登録したアカウントのパスワードハッシュを使用してください
- 新しいテストユーザーを登録後、データベースから `password_hash` を取得して追加してください

## パスワードハッシュの取得方法

```sql
-- 登録したユーザーのハッシュを確認
SELECT email, password_hash FROM users WHERE email = 'your-email@example.com';
```

## ファイルの追加方法

1. 画面から新しいテストユーザーを登録
2. 上記のSQLでパスワードハッシュを取得
3. `00X_insert_test_xxx.sql` という形式でファイルを作成
4. `ON CONFLICT (email) DO NOTHING` を使用して冪等性を確保
