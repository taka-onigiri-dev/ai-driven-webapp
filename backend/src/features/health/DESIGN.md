# Health Check Feature

## 概要

システムの稼働状態を確認するためのヘルスチェックエンドポイント。

## エンドポイント

```
GET /api/v1/health
```

## レスポンス

```json
{
  "status": "ok",
  "version": "0.1.0",
  "timestamp": "2025-12-29T10:00:00Z"
}
```

## ステータスコード

- 200 OK: システムが正常に稼働中

## 実装方針

- データベース接続は不要（軽量なチェック）
- 認証不要
- バージョン情報はCargo.tomlから取得
- タイムスタンプはUTCで返す

## テスト方針

- HTTPレスポンスの確認
- ステータスコード200の確認
- レスポンス形式の確認
